# source

## External Authorization

1. `extensions/filters/http/ext_authz/`
   1. **Envoyが外部認証サービスとどのように連携するかを定義している部分**です。
   2. 外部認証HTTPフィルターのコードが含まれており、Envoyの設定ファイルで定義された内容がどのように内部構造にマッピングされ、リクエストを外部サービスに送信し、レスポンスを処理するかを理解する最初のステップです。
2. `server/`
   1. Envoyの**メインサーバー構造、設定の読み込み、フィルターチェーンの管理**など、Envoy全体のライフサイクルを定義しています。
   2. 特に、**HTTPフィルター**がどのようにロードされ、リクエスト処理パイプラインに挿入されるかを理解するのに役立ちます。External AuthorizationはHTTPフィルターの一種です。
3. `common/`
   1. **HTTPメッセージの表現、ヘッダー操作、リクエスト/レスポンスのストリーミング**など、Envoyの基本的なネットワーク処理のロジックが含まれています。
   2. External Authorizationフィルターがリクエストのヘッダーやボディにアクセス・操作する際に、これらの共通ライブラリが利用されます。
4. `exe/`
   1. Envoyの**起動プロセス**と**メインエントリーポイント**が含まれています。
   2. これは、Envoyがどのように初期化され、`server/`で定義されたサーバーが起動されるかを見るのに役立ちますが、External Authorizationのロジック自体を理解するには優先度は低いです。
5. `docs/`
